# Comprehensive Traffic Accident Analysis Tool for Abu Dhabi 2025

## Overview
This tool provides a comprehensive analysis platform for the Traffic Accident Reports in Abu Dhabi Dataset 2025. It enables users to analyze accident-prone areas, peak accident times, and the impact of weather and road conditions on accident rates.

## Features
- Analyze most accident-prone areas (streets or cities).
- Determine peak traffic accident times.
- Identify common causes of accidents.
- Assess the impact of weather or road surface conditions.
- Interactive dashboards and data visualizations.

## Prerequisites
- Python 3.8 or above
- Libraries: pandas, matplotlib, seaborn
- Dataset: Traffic_Accident_Data_Abu_Dhabi_2025.xlsx

## Installation
1. Clone the repository:
    bash
    git clone https://github.com/YourUsername/TrafficAccidentAnalysis.git
    cd TrafficAccidentAnalysis
    
2. Install required Python libraries:
    bash
    pip install pandas matplotlib seaborn
    
3. Place the dataset file in the project directory.

## Usage
1. Run the script:
    bash
    python analysis_tool.py
    
2. View the visualizations and printed insights in the terminal.

## Example Outputs
- **Graph:** Number of accidents by month.
- **Graph:** Number of accidents by weekday.
- **List:** Top 10 accident-prone streets.
